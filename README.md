## System Description

The Crop Disease Prediction Expert System is a rule based system designed to diagnose diseases affecting selected crops based on user provided symptoms.

The system uses Prolog for logical reasoning, applying AND-based rules to match symptoms with predefined disease knowledge. A frontend interface allows users to input symptoms, while the backend connects the interface to the Prolog inference engine to generate accurate diagnoses.

The goal of the system is to provide an efficient and structured approach to identifying crop diseases.
